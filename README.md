# G.M. English School Website

A modern, responsive, and SEO-friendly website for G.M. English School built with React, TypeScript, Vite, and Tailwind CSS.

## Features

- Responsive Design
- Modern UI/UX
- SEO Optimized
- Admission Information
- Academics Section
- Faculty Information
- Infrastructure Gallery
- Contact Form
- Google Maps Integration
- Fast Loading
- Mobile Friendly

## Tech Stack

- React 
- TypeScript
- Vite
- Tailwind CSS
- React Router DOM
